# Firebase-Chat-App

Çevrimiçi chat uygulaması; kullanıcıların e-posta ile giriş yaparak sohbet yapabileceği bir uygulamadır. Firebase altyapısını kullanarak yapılmıştır. 

# Kullanılan Teknolojiler

- React
- Firebase:
- Firestore Database
- SCSS
- UUID
- React Icons
- Emoji Picker React
